# 커비샵 만들기 👛
React와 graphql로 만든 쇼핑몰 서비스입니다. 프로젝트 세팅은 Vite + React + Typescript로 진행,
React-Query와 Recoil을 이용하여 상태관리를 하였고, emotion을 이용한 Global UI 관리, 절대경로 설정을 진행하였습니다.
mock API를 생성하여 백엔드와 통신하는 것과 같이 데이터를 관리합니다.
